# Автоматизация тестирования API: ApiMocker

Данный проект представляет собой тестовый фреймворк для автоматизации тестирования REST API сервиса ApiMocker. Построен
на стеке Python + Pytest + Requests

## Технологии и версии

| Инструмент        | Версия  | Назначение                                 |
|:------------------|:--------|:-------------------------------------------|
| **Python**        | `3.10+` | Язык программирования                      |
| **Pytest**        | `7.4+`  | Фреймворк для организации и запуска тестов |
| **Requests**      | `2.31+` | HTTP-клиент для отправки запросов к API    |
| **Allure-pytest** | `2.13+` | Генерация детальных отчетов с вложениями   |

## Структура проекта

```text
api_mocker_qa/
│
├── allure-results/         # Временные отчеты (JSON, текстовые логи после тестов)
│
├── core/                   # Ядро фреймворка
│   └── api_client.py       # Базовый клиент (настройка requests и логирование)
│
├── services/               # Слой описания API (Service Objects)
│   ├── base_service.py     # Базовая логика и CRUD методы для всех ресурсов
│   ├── users_service.py    # Сервис для работы с эндпоинтами /users
│   ├── posts_service.py    # Сервис для работы с эндпоинтами /posts
│   ├── todos_service.py    # Сервис для работы с эндпоинтами /todos
│   └── comments_service.py # Сервис для работы с эндпоинтами /comments
│
├── tests/                  # Набор автоматизированных тестовых сценариев
│   ├── test_users.py       # Тесты пользователей
│   ├── test_posts.py       # Тесты постов
│   ├── test_todos.py       # Тесты задач
│   └── test_comments.py    # Тесты комментариев
│
├── venv/                   # Виртуальное окружение Python
│
├── .gitignore              # Исключения для Git (venv, отчеты, лог-файлы)
├── conftest.py             # Глобальные фикстуры и настройки окружения
├── pytest.ini              # Главный файл конфигурации Pytest (флаги запуска)
├── README.md               # Документация проекта
└── requirements.txt        # Список необходимых библиотек
```

## Запуск проекта

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url_вашего_репозитория>
   cd api_mocker_qa
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
    python -m venv venv
    # Windows:
    venv\Scripts\activate
    # macOS/Linux:
    source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
    pip install -r requirements.txt
   ```

## Запуск тестов

Все базовые настройки (папка отчетов, логирование) прописаны в `pytest.ini`

* **Запуск всех тестов:**
    ```bash
    pytest
    ```
* **Запуск конкретного сервиса:**
    ```bash
    pytest tests/test_posts.py
    ```
* **Запуск с подробным логированием запросов (флаг -s):**
    ```bash
    pytest -s
    ```

## Отчетность (Allure)

После прогона тестов результаты сохраняются в папку `allure-results`. Для просмотра интерактивного отчета выполните:

```bash
allure serve allure-results
```

## Реализованные сценарии

| Файл теста          | Описание сценария                                                                                                                                                       | 
|:--------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `test_users.py `    | Проверка операций пользователя, получение вложенных постов и задач по `userId `                                                                                         |
| `test_posts.py `    | Тестирование пагинации `(_page, _limit)`, создание постов и функционал добавления лайков.                                                                               |
| `test_todos.py `    | Фильтрация задач по владельцу и статусу completed (проверка логических фильтров `true/false`)                                                                           |
| `test_comments.py ` | Проверка поиска комментариев                                                                        названиях товаров                                                   |
